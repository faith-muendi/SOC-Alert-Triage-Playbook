# Phishing Triage Toolkit (Python)

## Overview
This repository contains a **Python-based phishing triage toolkit** designed to assist SOC analysts in analyzing suspected phishing emails.

The toolkit automates common Tier 1 phishing investigation tasks, helping analysts quickly identify malicious indicators and determine escalation requirements.

---

## Features
- URL extraction and analysis
- Domain reputation checks
- Email header inspection
- IOC extraction (URLs, IPs, domains)
- Structured output for SOC workflows

---

## Use Case
SOC analysts frequently receive phishing alerts that require repetitive checks across multiple tools.  
This toolkit centralizes those checks into a single workflow to improve efficiency and consistency.
